# minenohara-dic

## Install

### PyQt

/src/

```
$ sudo apt-get install qttools5-dev-tools
(env) $ pip install PyQt5
(env) $ pip install aiohttp
```

### Eel

/front/

```
(env) $ pip install eel
```

### Vuex + Eel

/dic/

```
$ vue create front
$ cd front
$ vue add vuetify
```

```
 $ cd front
 $ npm run serve
 $ npm run build
```

```
$ python3 -m venv env
$ . env/bin/activate
(env) $ pip install eel
```

```
(env) $ python gui.py
```
